# AllenDataHub Frontend

React + Vite + TypeScript frontend for AllenDataHub.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env.local` file:

```
VITE_BACKEND_URL=https://your-backend-url.com
```

## Deployment

Deployed on Vercel. Set the `VITE_BACKEND_URL` environment variable in Vercel dashboard to point to your backend API.
